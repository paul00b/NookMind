Interaction & Behavior

  • Swipe: Touch drag left/right to change slides, with momentum & snap
  • Skip: Top-right on every slide — jumps straight to login (or home if logged in)
  • Progress: Pill-shaped dots at bottom — active dot is wider + colored
  • Get Started: Only on last slide — goes to login (or home if already authenticated)
  • Persistence: localStorage flag after completion; replayable from Settings
  • i18n: All text in EN/FR translation files
  • Color progression: Amber → Indigo → Teal (matching media modes)
